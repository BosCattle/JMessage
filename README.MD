## 如何使用
### Gradle
```shell
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
	dependencies {
  	        compile 'com.github.BosCattle.JMessage:PowerSupportUI:Tag'
  	}
```
### Maven
```shell
<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
</repositories>
<dependency>
  	    <groupId>com.github.BosCattle.JMessage</groupId>
        <artifactId>PowerSupportUI</artifactId>
        <version>Tag</version>
 </dependency>
```
